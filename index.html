关于掩码

什么是标志flag

在掩码的语境下，标志flag，是只有1位的掩码，取值范围range(flag)range(\mathrm{flag})range(flag)满足：

range(flag)={0,1}range(\mathrm{flag}) = \{0, 1\} range(flag)={0,1}

。

flag可以是一个单纯的取值范围为{0,1}\{0,1\}{0,1}的二值的状态量，也可以是相对于另一个取值范围是{s1,s2}\{s_1, s_2\}{s1​,s2​}的二值的状态量state，所建立一一映射

flag:{s1,s2}↔{0,1}\mathrm{flag}: \{s_1, s_2\} \leftrightarrow \{0,1\} flag:{s1​,s2​}↔{0,1}

。

Note 更一般而且常见的，标志flag是一个离散的状态量，取值范围是自然数集N\NN的子集。它可以是单纯的状态量，也可以是：设存在一个状态量state，它的取值范围range(state)range(\mathrm{state})range(state)是离散的，flag是range(state)range(\mathrm{state})range(state)到自然数集N\NN的映射

flag:range(state)→N\mathrm{flag}: range(\mathrm{state}) \to \N flag:range(state)→N

。也即，使用不同的自然数来分别枚举和指代状态量state的各个状态值。

什么是掩码mask

掩码mask，是一个二进制的位模式，取值范围range(mask)range(\mathrm{mask})range(mask)满足：

range(mask)={0,1}n,n∈Nrange(\mathrm{mask}) = \{0, 1\}^n, n \in \N range(mask)={0,1}n,n∈N

。

mask是一个状态量，或者状态映射：给定状态向量state，是一组二值的状态量{statei∣0<i≤n,n∈N}\{\mathrm{state}_i \big| 0 < i \le n, n \in \N\}{statei​∣∣​0<i≤n,n∈N}的序列，分别有取值范围range(statei)={si,1,si,2}range(\mathrm{state}_i)=\{s_{i, 1}, s_{i, 2}\}range(statei​)={si,1​,si,2​}，mask是相对于state，所建立一一映射

mask:×ni=1{si,1,si,2}↔{0,1}n\mathrm{mask}: \underset{i=1}{\overset{n}{\times}}\{s_{i, 1}, s_{i, 2}\} \leftrightarrow \{0,1\}^n mask:i=1×n​​{si,1​,si,2​}↔{0,1}n

。

Note 标志flag的值域是标量scalar，掩码mask的值域是向量vector，两者都是状态量或者状态映射。
可以认为，mask是建立在{0,1}n,n∈N\{0,1\}^n,n \in \N{0,1}n,n∈N上的线性空间，给定它的一组基{ui∈{0,1}n∣0<i≤n;∀0<i,j≤n,ui[j]=1  ⟺  i=j}\{\bold{u}_i \in \{0,1\}^n \big| 0 < i \le n;\forall 0 < i, j \le n,\bold{u}_i[j] = 1 \iff i = j\}{ui​∈{0,1}n∣∣​0<i≤n;∀0<i,j≤n,ui​[j]=1⟺i=j}，也即分别有ui\bold{u}_iui​对应于mask的第iii位。令标志flagi\mathrm{flag}_iflagi​对应于mask\mathrm{mask}mask的第iii位的值，即有flagi=mask[i]\mathrm{flag}_i = \mathrm{mask}[i]flagi​=mask[i]且有线性组合

mask=∑i=1nflagi⋅ui\mathrm{mask} = \sum_{i=1}^n \mathrm{flag}_i \cdot \bold{u}_i mask=i=1∑n​flagi​⋅ui​

。总而言之，mask的每一位分别对应一个flag，mask是多个占据不同位的flag的线性组合，因此mask是一个状态向量或者组合状态映射。

利用异常实现越级return（Python实现）

异常是一类消息

在面向对象的世界里，对象之间的交互表现为消息的传递。异常也是一种消息，就我的理解，异常并非错误，它经常用来表明存在错误，但本质上它是一类特殊的消息，关联着一种跳出局部范围的机制。而在Python的世界里，所有异常都必须是BaseException的子类。
所谓异常是消息，指的是一个异常对象可以携带信息，捕获异常（就得到了一个异常对象）或者传递一个异常对象，你就可以从这个异常对象中提取信息。这个消息有时会被认为是一个指令或者信号，即便仅仅只是捕获到了一个特定类型的异常对象，例如，某个无限循环中有一个捕获异常的语句，当捕获到某个类型的异常对象时，它就认为接收到了退出的信号，快速、优雅地退出。
